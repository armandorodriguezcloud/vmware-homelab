# VMware Virtualization Homelab

> A sandbox for testing infrastructure designs safely before production.

## Overview

A multi-host VMware lab with ESXi hosts under vCenter running HA, DRS, and vMotion, segmented virtual networking, and shared storage. Used to test designs, snapshots, and recovery scenarios safely.

## Tech stack

- VMware vSphere
- ESXi
- vCenter
- Networking
- Storage

## What this lab demonstrates

- ESXi hosts managed under vCenter
- HA, DRS, and vMotion
- Segmented virtual networking
- Shared storage
- Snapshot and recovery testing
